## Описание модулей и функций

### Модуль Product

Содержит методы и свойства класса **Product**\
**Атрибуты класса:**

- **name** - название,
- **description** - описание,
- **__price** - цена (приватный атрибут),
- **quantity** - количество в наличии.

#### Методы класса Product
#### Магические методы:
- **__str\__** - для строкового отображения класса **Product** в виде:\
<Название продукта>, <Цена> руб. Остаток: <Х> шт.
- **__add\__** - возвращает сумму произведений цены на количество у двух продуктов.
#### Методы класса:
- **price** - метод-геттер для доступа к цене продукта 

- **price** - метод-сеттер для изменения цены продукта.\
Проверяет значение новой цены: если цена отрицательная,\
выводится соответствующее сообщение в консоль и цена не изменяется.\
Нулевую цену установить можно.\
Если цена товара понижается, пользователю необходимо подтвердить изменение вручную:
  - **"Y"** ("yes") - для согласия понизить цену
  - **"N"** ("no") - для отмены действия (цена остаётся неизменной).

- **new_product** - класс-метод для создания экземпляра класса **Product**.\
Принимает на вход параметры товара в виде словаря и возвращает созданный объект класса 
Product.\
 Метод проверяет наличие товара (объекта Product) с таким же именем:
  - если такой уже существует, складывается количество имеющегося в наличии товара и новое количество.
  - при конфликте цен выбирается та, которая является более высокой
#### Для хранения всех создаваемых объектов класса Product используется приватное свойство **__product_list**
Алгоритм работы с классом **Product** предусматривает создание экземпляра класса **без привязки к какой-либо категории.**\
Класс-метод создания нового экземпляра класса создан с учётом такого принципа создания объектов класса **Product**.

### Модуль Category

Содержит методы и свойства класса **Category**\
**Атрибуты класса:**

- **name** - название,
- **description** - описание,
- **__products** - приватный, содержит список товаров в категории - экземпляров класса Product.

Доступные для каждого объекта класса атрибуты\
с общей информацией для всех объектов:

- **category_count** - количество категорий,
- **product_count** - количество товаров (продуктов для каждой категории).
#### Методы класса Category
#### Магический метод **__str\__** 
- для строкового отображения класса **Category** в виде:\
<Название категории>, количество продуктов: <Х> шт.\
Количество продуктов считается из общего числа всех продуктов в списке товаров категории.
Общее количество складывается из атрибута "Количество" (quantity) каждого товара.
#### Методы класса
 - **products** - метод-геттер возвращает список товаров в виде строк в формате:\
"<Название продукта>, 80 руб. Остаток: 15 шт."
 - **add_product** - добавляет продукт в список товаров в категории - в атрибут **products**
 - **product_in_category** - метод-геттер для доступа к списку продуктов в категории (атрибуту **__products**)

### Модуль utils

Загружает данные по категориям и товарам из файла JSON.\
Содержит функции:\
**read_json** - читает данные из JSON-файла и преобразует их в словарь (dict).\
Если файл не найден, возвращает пустой словарь.

**create_objects_from_json** - на основе чтения данных из словаря на входе функции создаёт объекты -
экземпляры классов Category и Product.\

Файл JSON **"products.json"**, из которого загружаются данные для созданий категорий и товаров, находится в каталоге **data**.

## Тестирование (Pytest)

### Тестирование классов и их методов выполняется в файле "test_classes.py"

Тестируются:
 - Конструкторы классов **Product** и **Category**
 - Корректность подсчёта количества категорий и продуктов для класса **Category**
 - Добавление продукта в список товаров в отдельной категории
 - Выод списка товаров в категории
 - Создание экземпляра класса **Product** и изменение количества и цены при создании товара с таким же названием\
 - Строковые представления для классов **Product** и **Category**
 - Магический метод **__add\__** для сложения 2-х товаров

       * Примечание: для модуля utils тесты не созданы 
#### В файле "conftest.py" находятся фикстуры с экземплярами классов, словарями для тестирования.
